Choose a Simulation Mode

Our system includes three simulation modes: Single Sawmill, Multi Sawmill, and All Sawmills. Use these modes to explore scenarios, compare mills, and analyze network-wide performance. You can modify data before running any simulation.

Single Sawmill Simulation

Run this to visualize truck routing for one sawmill and its associated logging sites.

  • Focus on a single mill and its connected logging sites.
  • Update logging-site data, add new sites, or remove existing ones to see their impact.
  • Adjust sawmill parameters as needed.
  • Handles crane breakdowns by intelligently redirecting trucks to alternative mills.
  • Processes related mills/sites behind the scenes for transfers; results are presented for your selected mill.
Start Single Sawmill →

Multi Sawmill Simulation

Run this to compare strategies and interactions across multiple selected mills simultaneously.

  • Select two or more mills to run concurrently.
  • Edit parameters across the selected mills to compare routing and capacity strategies.
  • View results for all selected mills after the simulation completes.
Start Multi Sawmill →

All Sawmills Simulation

Choose this to establish a system-wide baseline or to stress-test all 59 sawmills at once.

  • Run the entire network to assess throughput, bottlenecks, and inter-mill dependencies.
  • Support for data edits before execution.
  • Captures cross-mill effects such as breakdown-driven truck transfers at scale.
  • Best for strategic planning, parameter sweeps, and network-wide performance baselining.
Start All Sawmills Simulation →
